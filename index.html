Встроенные HTML-элементы

HTML определяет длинный список доступных встроенных тегов, полный список которых можно найти на Сеть разработчиков Mozilla.

  • Для выделения текста жирным шрифтом используйте <strong>.
  • Чтобы выделить текст курсивом, используйте <em>.
  • В аббревиатурах, таких как HTML следует использовать <abbr>, с необязательным заголовок атрибут для полной фразы.
  • В цитатах, таких как — Марк Отто, следует использовать <cite>.
  • Удаленный текст должен использовать <del> и вставленный текст должен использовать <ins>.
  • Надстрочный текст использует <sup> и подстрочный текст использует <sub>.

Большинство из этих элементов оформлены браузерами с небольшими изменениями с нашей стороны.
